# semantic_similarity #
## Installation ##
* ### docker ###
  1. Pull the docker container from: https://hub.docker.com/repository/docker/liaimskye/semantic_task/general
  2. In the command terminal run the docker container.

* ### Github ###
  1. clone the repository to your computer.
  2. open the command terminal in the project directory and run 'pip install -r requirements.txt'
  3. Run the semantic.py file
